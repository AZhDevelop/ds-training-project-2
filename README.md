# Проект 2: Обзор рынка Food and Beverage

Этот проект сосредоточен на анализе и исследовании данных, связанных с рынком общепита. Он содержит различные наборы данных и скрипты анализа для обзора рынка.

## Как запустить проект
Убедитесь, что у вас установлены все необходимые библиотеки, перечисленные в файле requirements.txt.

```bash
pip install -r requirements.txt
```

В проекте использовалось свое ядро Jupyter для работы с окружением и зависимостями из файла requirements.txt. Для использования такого ядра нужно зарегистрировать это окружение в Jupyter под понятным именем, например:

``` bash
python -m ipykernel install --user --name=my-venv-kernel --display-name "Python (My Project)"
```

## Файлы данных

- `rest_info.csv`: Данные информации.
- `rest_price.csv`: Данные цен.

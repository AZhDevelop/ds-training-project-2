# Проект 2: Обзор рынка Food and Beverage

Этот проект сосредоточен на анализе и исследовании данных, связанных с рынком общепита. Он содержит различные наборы данных и скрипты анализа для обзора рынка.

## Зависимости

Для запуска этого проекта вам необходимо установить требуемые пакеты Python, перечисленные в `requirements.txt`.

1.  **Клонируйте репозиторий (если применимо).**
2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

## Файлы данных

- `rest_info.csv`: Данные информации.
- `rest_price.csv`: Данные цен.
- `project-2-fb-market-overview.ipynb`: Jupyter Notebook для анализа.

## Использование

Запустите ноутбук `project-2-fb-market-overview.ipynb`, чтобы начать анализ.
